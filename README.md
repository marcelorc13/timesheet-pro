# TimeSheet PRO
 *TimeSheet PRO* é um sistema de gerenciamento de horas e ponto digital criado para simplificar o controle de jornada de trabalho em empresas de pequeno e médio porte. Nosso objetivo é automatizar tarefas manuais, eliminar erros em planilhas e fornecer uma plataforma confiável e centralizada para gestores e colaboradores.

### Recursos Principais
- Registro de Ponto Ágil: Colaboradores podem registrar sua entrada e saída com um único clique.

- Cálculo Automático: O sistema calcula automaticamente horas trabalhadas, horas extras e atrasos.

- Painel de Gestão para RH: Visualize o status de toda a equipe e acesse relatórios de ponto detalhados.

- Automação de Endereços: Integração com a API pública do ViaCEP para preenchimento automático de endereços, garantindo agilidade e precisão.
